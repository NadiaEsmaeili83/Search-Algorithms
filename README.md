# 🤖 جستجوی هوش مصنوعی - الگوریتم‌های جستجو

پیاده‌سازی الگوریتم‌های جستجوی **BFS**، **DFS**، **UCS** و **A\*** در یک محیط گرید 15x15 با موانع ثابت و متحرک.

---

## 📚 اطلاعات دانشجو
- **نام**: نادیا اسماعیلی
- **شماره دانشجویی**: 402103231
- **درس**: مبانی هوش مصنوعی

---

## 🎯 الگوریتم‌های پیاده‌سازی شده

| الگوریتم | توضیح |
|----------|-------|
| **BFS** | جستجوی سطح اول (مختص بهینه نیست) |
| **DFS** | جستجوی عمق اول (سریع ولی غیر بهینه) |
| **UCS** | جستجوی یکنواخت (هزینه بهینه) |
| **A\* (منهتن)** | جستجوی با هیوریستیک منهتن |
| **A\* (اقلیدسی)** | جستجوی با هیوریستیک اقلیدسی |
